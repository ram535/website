Uninitialized variables

Uninitialized variables have an initial value of null. Even variables with numeric types are initially null, because numbers are objects.

  void main() {
    var a;
    num b;
    int c;
    double d;
    bool e;

    print(a); // null
    print(b); // null
    print(c); // null
    print(d); // null
    print(e); // null
  }
